# Trust API

A Go-based backend service that analyzes suspicious text, URLs, and screenshots, then returns confidence levels, risk signals, and recommended actions for safer content handling.

## Goals
- Analyze suspicious digital content
- Return confidence and risk signals
- Help apps decide whether content should be trusted, labeled, or warned on
- Serve as a learning project with startup potential

## MVP Scope
- Text input
- URL input
- Image/screenshot input

## Planned Output
- Claim summary
- Source type
- Risk reasons
- Confidence level
- Recommended UI action

## Tech Stack
- Go
- PostgreSQL
- Redis later if needed
